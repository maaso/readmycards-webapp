ReadMyCards.eu

This website will read the contents of your smartcards, using any reader.
The service is completely free for personal use, and we never save any of the data.

Download GCL
To use this website, you will need to install a small piece of software called the Generic Connector Library (GCL). This will enable us to communicate with your card reader, no matter which browser you use.
